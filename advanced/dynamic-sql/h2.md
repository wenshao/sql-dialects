```sql
PREPARE stmt FROM 'SELECT * FROM users WHERE id = ?';
EXECUTE stmt USING 42;
DEALLOCATE stmt;
```

## Java 存储过程中的动态 SQL

H2 支持 Java 编写的存储过程
CREATE ALIAS DYNAMIC_QUERY AS $$
void dynamicQuery(Connection conn, String table) throws SQLException {
Statement stmt = conn.createStatement();
ResultSet rs = stmt.executeQuery("SELECT * FROM " + table);
}
$$;
CALL DYNAMIC_QUERY('users');

## JDBC 替代方案 (Java)

PreparedStatement ps = conn.prepareStatement("SELECT * FROM users WHERE age > ?");
ps.setInt(1, 18);
ResultSet rs = ps.executeQuery();
注意：H2 支持基本的 PREPARE / EXECUTE
注意：复杂动态 SQL 通过 Java 存储过程或 JDBC 实现
限制：不支持 EXECUTE IMMEDIATE
限制：无 PL/SQL 或 PL/pgSQL 过程语言
