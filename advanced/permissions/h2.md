# H2: 权限管理

## H2 支持基本的用户和权限管理

## 用户管理


## 创建用户

```sql
CREATE USER alice PASSWORD 'password123';
CREATE USER IF NOT EXISTS bob PASSWORD 'password456';
```

## 管理员用户

```sql
CREATE USER admin PASSWORD 'admin123' ADMIN;
```

## 修改密码

```sql
ALTER USER alice SET PASSWORD 'new_password';
```

## 删除用户

```sql
DROP USER alice;
DROP USER IF EXISTS alice;
```

## 角色


```sql
CREATE ROLE app_readonly;
CREATE ROLE app_readwrite;

GRANT app_readonly TO alice;
GRANT app_readwrite TO bob;

DROP ROLE app_readonly;
```

## 授权


## 表权限

```sql
GRANT SELECT ON users TO alice;
GRANT SELECT, INSERT, UPDATE, DELETE ON users TO bob;
GRANT ALL ON users TO admin;
```

## Schema 权限

```sql
GRANT SELECT ON SCHEMA PUBLIC TO alice;
```

## 撤销权限

```sql
REVOKE INSERT ON users FROM bob;
REVOKE ALL ON users FROM alice;
```

## 查看权限


```sql
SELECT * FROM INFORMATION_SCHEMA.TABLE_PRIVILEGES WHERE GRANTEE = 'ALICE';
SELECT * FROM INFORMATION_SCHEMA.ROLES;
```

## 特殊设置


允许所有（禁用权限检查，仅开发/测试用）
连接 URL: jdbc:h2:mem:test;MODE=...
管理员权限

```sql
GRANT ALTER ANY SCHEMA TO admin;
```

注意：H2 权限管理较为简单
注意：ADMIN 关键字创建管理员用户
注意：内存数据库重启后用户和权限丢失
注意：不支持列级权限
注意：不支持行级安全（RLS）
