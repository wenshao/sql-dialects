# H2: 触发器

H2 触发器通过 Java 类实现
============================================================
创建触发器（Java 实现）
============================================================
BEFORE INSERT
Java 类：
public class BeforeInsertTrigger implements Trigger {
public void fire(Connection conn, Object[] oldRow, Object[] newRow) {
newRow[4] = new Timestamp(System.currentTimeMillis()); // updated_at
}
public void init(Connection conn, String schema, String trigger,
String table, boolean before, int type) {}
public void close() {}
public void remove() {}
}

```sql
CREATE TRIGGER trg_users_before_insert
BEFORE INSERT ON users
FOR EACH ROW
CALL "com.example.BeforeInsertTrigger";
```

## AFTER INSERT

```sql
CREATE TRIGGER trg_users_after_insert

FOR EACH ROW
CALL "com.example.AfterInsertTrigger";
```

## 使用 CREATE ALIAS + TRIGGER

```sql
CREATE ALIAS AUDIT_INSERT AS $$
void auditInsert(Connection conn, Object[] newRow) throws Exception {
    conn.createStatement().execute(
        "INSERT INTO audit_log (action, record_id, created_at) VALUES ('INSERT', " +
        newRow[0] + ", CURRENT_TIMESTAMP)"
    );
}
$$;
```

## 注意：这种方式需要使用 source 模式

## 语句级触发器


```sql
CREATE TRIGGER trg_users_after_stmt
AFTER INSERT ON users

```

## 管理触发器


```sql
DROP TRIGGER IF EXISTS trg_users_before_insert;

SELECT * FROM INFORMATION_SCHEMA.TRIGGERS WHERE TABLE_NAME = 'USERS';
```

注意：H2 触发器必须实现 org.h2.api.Trigger 接口
注意：支持 BEFORE/AFTER、INSERT/UPDATE/DELETE
注意：支持行级和语句级触发器
注意：Java 类需要在 classpath 中
