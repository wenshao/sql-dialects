# H2: ALTER TABLE

> 参考资料:
> - [H2 SQL Reference - Commands](https://h2database.com/html/commands.html)
> - [H2 - Data Types](https://h2database.com/html/datatypes.html)
> - [H2 - Functions](https://h2database.com/html/functions.html)
> - 添加列

```sql
ALTER TABLE users ADD COLUMN phone VARCHAR(20);
ALTER TABLE users ADD COLUMN IF NOT EXISTS phone VARCHAR(20);
```

## 添加列带约束

```sql
ALTER TABLE users ADD COLUMN status INT NOT NULL DEFAULT 1;
```

## 删除列

```sql
ALTER TABLE users DROP COLUMN phone;
ALTER TABLE users DROP COLUMN IF EXISTS phone;
```

## 修改列类型

```sql
ALTER TABLE users ALTER COLUMN age SET DATA TYPE BIGINT;
```

## 修改列名

```sql
ALTER TABLE users ALTER COLUMN email RENAME TO email_address;
ALTER TABLE users RENAME COLUMN email TO email_address;
```

## 修改默认值

```sql
ALTER TABLE users ALTER COLUMN status SET DEFAULT 0;
ALTER TABLE users ALTER COLUMN status DROP DEFAULT;
```

## 修改 NOT NULL

```sql
ALTER TABLE users ALTER COLUMN email SET NOT NULL;
ALTER TABLE users ALTER COLUMN email DROP NOT NULL;
```

## 添加约束

```sql
ALTER TABLE users ADD CONSTRAINT uq_email UNIQUE (email);
ALTER TABLE users ADD CONSTRAINT chk_age CHECK (age > 0);
ALTER TABLE users ADD PRIMARY KEY (id);
ALTER TABLE orders ADD CONSTRAINT fk_user FOREIGN KEY (user_id) REFERENCES users(id);
```

## 删除约束

```sql
ALTER TABLE users DROP CONSTRAINT uq_email;
ALTER TABLE users DROP CONSTRAINT IF EXISTS uq_email;
ALTER TABLE orders DROP CONSTRAINT fk_user;
```

## 重命名表

```sql
ALTER TABLE users RENAME TO members;
```

## 修改 IDENTITY 列

```sql
ALTER TABLE users ALTER COLUMN id RESTART WITH 1000;
```

## 添加自增

```sql
ALTER TABLE users ALTER COLUMN id SET GENERATED ALWAYS AS IDENTITY;
```

## 修改列顺序（H2 特有）

```sql
ALTER TABLE users ALTER COLUMN phone SET BEFORE email;
ALTER TABLE users ALTER COLUMN phone SET AFTER username;
```

注意：H2 支持大部分标准 ALTER TABLE 操作
注意：修改列顺序（SET BEFORE/AFTER）是 H2 特有功能
注意：ALTER COLUMN 可以同时修改类型和约束
注意：在兼容模式下，语法可能略有不同
