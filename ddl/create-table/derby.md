# Derby: CREATE TABLE

> 参考资料:
> - [Derby SQL Reference](https://db.apache.org/derby/docs/10.16/ref/)
> - [Derby Developer Guide](https://db.apache.org/derby/docs/10.16/devguide/)
> - Apache Derby 是 Java 嵌入式数据库（也称 Java DB）
> - 基本建表

```sql
CREATE TABLE users (
    id         INT NOT NULL GENERATED ALWAYS AS IDENTITY (START WITH 1, INCREMENT BY 1),
    username   VARCHAR(64) NOT NULL,
    email      VARCHAR(128) NOT NULL,
    age        INT,
    balance    DECIMAL(10,2) DEFAULT 0.00,
    bio        CLOB,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    PRIMARY KEY (id)
);
```

## GENERATED BY DEFAULT（允许手动指定值）

```sql
CREATE TABLE orders (
    id         INT NOT NULL GENERATED BY DEFAULT AS IDENTITY,
    user_id    INT NOT NULL,
    amount     DECIMAL(10,2),
    order_date DATE,
    status     INT DEFAULT 0,
    PRIMARY KEY (id)
);
```

## BIGINT 自增

```sql
CREATE TABLE events (
    id         BIGINT NOT NULL GENERATED ALWAYS AS IDENTITY,
    event_type VARCHAR(50),
    payload    VARCHAR(32672),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    PRIMARY KEY (id)
);
```

## 完整约束

```sql
CREATE TABLE employees (
    id         INT NOT NULL GENERATED ALWAYS AS IDENTITY,
    name       VARCHAR(100) NOT NULL,
    email      VARCHAR(128) NOT NULL,
    dept_id    INT,
    salary     DECIMAL(10,2),
    PRIMARY KEY (id),
    UNIQUE (email),
    CHECK (salary >= 0),
    CONSTRAINT fk_dept FOREIGN KEY (dept_id) REFERENCES departments(id)
);
```

## CREATE TABLE AS SELECT（需要带列定义）

```sql
CREATE TABLE users_backup AS SELECT * FROM users WITH NO DATA;
INSERT INTO users_backup SELECT * FROM users WHERE age > 18;
```

## 临时表

```sql
DECLARE GLOBAL TEMPORARY TABLE temp_results (
    id   INT,
    val  VARCHAR(100)
) NOT LOGGED;
```

## 带数据类型

```sql
CREATE TABLE all_types (
    c_smallint  SMALLINT,
    c_int       INT,
    c_bigint    BIGINT,
    c_decimal   DECIMAL(10,2),
    c_real      REAL,
    c_double    DOUBLE,
    c_float     FLOAT,
    c_char      CHAR(10),
    c_varchar   VARCHAR(255),
    c_long_vc   LONG VARCHAR,
    c_clob      CLOB,
    c_blob      BLOB,
    c_date      DATE,
    c_time      TIME,
    c_timestamp TIMESTAMP,
    c_boolean   BOOLEAN
);
```

注意：Derby 的 IDENTITY 语法比较冗长
注意：VARCHAR 最大长度为 32672 字节
注意：不支持 CREATE OR REPLACE TABLE
注意：不支持 IF NOT EXISTS（需要捕获异常）
注意：临时表使用 DECLARE GLOBAL TEMPORARY TABLE
注意：Derby 同时支持嵌入式和网络服务器模式
