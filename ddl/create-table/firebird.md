# Firebird: CREATE TABLE

> 参考资料:
> - [Firebird SQL Reference](https://firebirdsql.org/en/reference-manuals/)
> - [Firebird Release Notes](https://firebirdsql.org/file/documentation/release_notes/html/en/4_0/rlsnotes40.html)

```sql
CREATE TABLE users (
    id         BIGINT        GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    username   VARCHAR(64)   NOT NULL,
    email      VARCHAR(255)  NOT NULL,
    age        INTEGER,
    balance    DECIMAL(10,2) DEFAULT 0.00,
    bio        BLOB SUB_TYPE TEXT,
    created_at TIMESTAMP     DEFAULT CURRENT_TIMESTAMP NOT NULL,
    updated_at TIMESTAMP     DEFAULT CURRENT_TIMESTAMP NOT NULL,
    CONSTRAINT uk_username UNIQUE (username),
    CONSTRAINT uk_email UNIQUE (email)
);
```

## Pre-3.0: use generators (sequences) + triggers for auto-increment

```sql
CREATE GENERATOR gen_users_id;
SET GENERATOR gen_users_id TO 0;

CREATE TABLE users_legacy (
    id       INTEGER NOT NULL PRIMARY KEY,
    username VARCHAR(64) NOT NULL
);

SET TERM !! ;
CREATE TRIGGER trg_users_legacy_bi FOR users_legacy
ACTIVE BEFORE INSERT POSITION 0
AS
BEGIN
    IF (NEW.id IS NULL) THEN
        NEW.id = GEN_ID(gen_users_id, 1);
END!!
SET TERM ; !!
```

## Sequence (3.0+ preferred syntax)

```sql
CREATE SEQUENCE seq_orders;
```

## 4.0+: IDENTITY columns

```sql
CREATE TABLE orders (
    order_id   BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    user_id    BIGINT NOT NULL,
    amount     DECIMAL(12,2),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

## Character sets

```sql
CREATE TABLE international_data (
    id      INTEGER NOT NULL PRIMARY KEY,
    name    VARCHAR(100) CHARACTER SET UTF8,
    notes   VARCHAR(500) CHARACTER SET WIN1252
);
```

## Computed columns

```sql
CREATE TABLE products (
    price     DECIMAL(10,2),
    tax_rate  DECIMAL(4,2),
    total     COMPUTED BY (price * (1 + tax_rate))
);
```

## BLOB types

```sql
CREATE TABLE files (
    id       INTEGER NOT NULL PRIMARY KEY,
    content  BLOB SUB_TYPE TEXT,           -- text BLOB
    data     BLOB SUB_TYPE BINARY,         -- binary BLOB (default)
    segment  BLOB SUB_TYPE 0 SEGMENT SIZE 4096
);
```

## Global temporary table (3.0+)

```sql
CREATE GLOBAL TEMPORARY TABLE gtt_results (
    id    BIGINT,
    value DECIMAL(10,2)
) ON COMMIT PRESERVE ROWS;
```

## Domain-based columns (Firebird domains are like custom types)

```sql
CREATE DOMAIN d_username AS VARCHAR(64) NOT NULL;
CREATE DOMAIN d_email AS VARCHAR(255)
    CHECK (VALUE CONTAINING '@');
CREATE DOMAIN d_positive_int AS INTEGER
    CHECK (VALUE > 0);

CREATE TABLE domain_example (
    id       INTEGER NOT NULL PRIMARY KEY,
    username d_username,
    email    d_email,
    quantity d_positive_int
);
```

## 3.0+: BOOLEAN type

```sql
CREATE TABLE flags (
    id     INTEGER NOT NULL PRIMARY KEY,
    active BOOLEAN DEFAULT FALSE
);
```

## External table (read from file)

```sql
CREATE TABLE ext_data (
    line CHAR(200)
) EXTERNAL FILE '/tmp/data.txt';
```

## RECREATE (drops and recreates if exists)

```sql
RECREATE TABLE temp_work (
    id    INTEGER NOT NULL PRIMARY KEY,
    data  VARCHAR(200)
);
```
