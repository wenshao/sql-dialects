## H2 是 Java 嵌入式数据库，支持多种兼容模式

基本建表

```sql
CREATE TABLE users (
    id         INT PRIMARY KEY AUTO_INCREMENT,
    username   VARCHAR(64) NOT NULL,
    email      VARCHAR(128) NOT NULL,
    age        INT,
    balance    DECIMAL(10,2) DEFAULT 0.00,
    bio        CLOB,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

## IDENTITY 列（自增主键）

```sql
CREATE TABLE orders (
    id         IDENTITY,                       -- 等同于 BIGINT AUTO_INCREMENT PRIMARY KEY
    user_id    INT NOT NULL,
    amount     DECIMAL(10,2),
    order_date DATE,
    status     INT DEFAULT 0
);
```

## GENERATED ALWAYS AS IDENTITY

```sql
CREATE TABLE products (
    id         INT GENERATED ALWAYS AS IDENTITY,
    name       VARCHAR(100) NOT NULL,
    price      DECIMAL(10,2) NOT NULL,
    category   VARCHAR(50)
);
```

## 序列 + 默认值

```sql
CREATE SEQUENCE user_seq START WITH 1 INCREMENT BY 1;
CREATE TABLE members (
    id         INT DEFAULT NEXT VALUE FOR user_seq PRIMARY KEY,
    name       VARCHAR(100)
);
```

## IF NOT EXISTS

```sql
CREATE TABLE IF NOT EXISTS logs (
    id         IDENTITY,
    level      VARCHAR(10),
    message    VARCHAR(1000),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

## CREATE TABLE AS SELECT

```sql
CREATE TABLE users_backup AS SELECT * FROM users WHERE age > 18;
```

CREATE OR REPLACE
H2 不直接支持 CREATE OR REPLACE TABLE，需先 DROP
临时表

```sql
CREATE LOCAL TEMPORARY TABLE temp_data (
    id   INT,
    val  VARCHAR(100)
);
```

## 内存表（不持久化）

```sql
CREATE MEMORY TABLE cache_data (
    key    VARCHAR(100) PRIMARY KEY,
    value  VARCHAR(1000)
);
```

## 带约束的完整建表

```sql
CREATE TABLE employees (
    id         INT PRIMARY KEY AUTO_INCREMENT,
    name       VARCHAR(100) NOT NULL,
    email      VARCHAR(128) UNIQUE,
    dept_id    INT,
    salary     DECIMAL(10,2) CHECK (salary >= 0),
    CONSTRAINT fk_dept FOREIGN KEY (dept_id) REFERENCES departments(id)
);
```

## 计算列

```sql
CREATE TABLE products_v2 (
    id         INT PRIMARY KEY AUTO_INCREMENT,
    price      DECIMAL(10,2),
    tax_rate   DECIMAL(5,4),
    total      DECIMAL(10,2) GENERATED ALWAYS AS (price * (1 + tax_rate))
);
```

注意：H2 支持内存模式（jdbc:h2:mem:dbname）和文件模式
注意：IDENTITY 是 BIGINT AUTO_INCREMENT PRIMARY KEY 的简写
注意：支持 PostgreSQL, MySQL, Oracle, SQL Server 兼容模式
注意：H2 支持 CLOB、BLOB 等大对象类型
