## е…ій”®е·®ејӮ

- **MySQL** дёҚж”ҜжҢҒ CREATE SEQUENCEпјҢдҫқиө– AUTO_INCREMENTпјҲдёҖеј иЎЁеҸӘиғҪжңүдёҖдёӘпјү
- **PostgreSQL 10+** жҺЁиҚҗ GENERATED AS IDENTITY жӣҝд»Ј SERIALпјҲSQL ж ҮеҮҶпјү
- **Oracle 12c+** еҗҢж—¶ж”ҜжҢҒ SEQUENCE е’Ң IDENTITY еҲ—пјҢGENERATED BY DEFAULT ON NULL жңҖзҒөжҙ»
- **SQL Server** зҡ„ IDENTITY жҳҜдј з»ҹж–№ејҸпјҢSEQUENCEпјҲ2012+пјүжӣҙзҒөжҙ»пјҢж”ҜжҢҒи·ЁиЎЁе…ұдә«
- **SQLite** дҪҝз”Ё INTEGER PRIMARY KEY йҡҗејҸиҮӘеўһпјҢAUTOINCREMENT дҝқиҜҒдёҚеӨҚз”Ё ID
- **MariaDB 10.3+** жҳҜ MySQL еҲҶж”Ҝдёӯе”ҜдёҖж”ҜжҢҒ CREATE SEQUENCE зҡ„
- **ClickHouse** дёҚж”ҜжҢҒеәҸеҲ—жҲ–иҮӘеўһпјҢжҺЁиҚҗдҪҝз”Ё UUIDпјҲgenerateUUIDv4/v7пјү
- **BigQuery/Hive/Spark/Flink** зӯүеӨ§ж•°жҚ®еј•ж“ҺдёҚж”ҜжҢҒеәҸеҲ—пјҢдҪҝз”Ё UUID жҲ– ROW_NUMBER
- **CockroachDB** зҡ„ SERIAL й»ҳи®ӨдҪҝз”Ё unique_rowid()пјҲйқһиҮӘеўһпјүпјҢдёҺ PostgreSQL дёҚеҗҢ
