# Derby: INSERT

> 参考资料:
> - [Derby SQL Reference](https://db.apache.org/derby/docs/10.16/ref/)
> - [Derby Developer Guide](https://db.apache.org/derby/docs/10.16/devguide/)
> - 单行插入

```sql
INSERT INTO users (username, email, age) VALUES ('alice', 'alice@example.com', 25);
```

## 多行插入（Derby 10.10+）

```sql
INSERT INTO users (username, email, age) VALUES
    ('alice', 'alice@example.com', 25),
    ('bob', 'bob@example.com', 30),
    ('charlie', 'charlie@example.com', 35);
```

## IDENTITY 列省略（GENERATED ALWAYS）

```sql
INSERT INTO users (username, email, age) VALUES ('alice', 'alice@example.com', 25);
```

## GENERATED BY DEFAULT（允许手动指定）

```sql
INSERT INTO orders (id, user_id, amount) VALUES (100, 1, 99.99);
```

## 从查询结果插入

```sql
INSERT INTO users_archive (username, email, age)
SELECT username, email, age FROM users WHERE age > 60;
```

## DEFAULT 值

```sql
INSERT INTO users (username, email) VALUES ('alice', 'alice@example.com');
```

## NULL 值

```sql
INSERT INTO users (username, email, age) VALUES ('alice', 'alice@example.com', NULL);
```

## 带 DEFAULT 关键字

```sql
INSERT INTO events (event_type, status) VALUES ('login', DEFAULT);
```

## 子查询插入

```sql
INSERT INTO high_value_users (username, email)
SELECT username, email FROM users
WHERE id IN (SELECT user_id FROM orders GROUP BY user_id HAVING SUM(amount) > 10000);
```

## 批量导入（使用系统过程）


## 使用 SYSCS_UTIL.SYSCS_IMPORT_TABLE 导入 CSV

```sql
CALL SYSCS_UTIL.SYSCS_IMPORT_TABLE(
    'APP',                           -- schema
    'USERS',                         -- 表名
    '/path/to/users.csv',            -- 文件路径
    ',',                             -- 列分隔符
    '"',                             -- 字符串分隔符
    'UTF-8',                         -- 字符集
    0                                -- 是否替换（0=插入，1=替换）
);
```

## 导入指定列

```sql
CALL SYSCS_UTIL.SYSCS_IMPORT_DATA(
    'APP', 'USERS',
    'USERNAME,EMAIL,AGE',            -- 插入列
    '1,2,3',                         -- 文件列位置
    '/path/to/users.csv',
    ',', '"', 'UTF-8', 0
);
```

## 导出


```sql
CALL SYSCS_UTIL.SYSCS_EXPORT_TABLE(
    'APP', 'USERS',
    '/path/to/export.csv',
    ',', '"', 'UTF-8'
);
```

注意：GENERATED ALWAYS 列不能手动指定值
注意：GENERATED BY DEFAULT 列允许手动指定值
注意：多行 VALUES 在较新版本（10.10+）才支持
注意：批量导入使用 SYSCS_UTIL 系统过程
注意：不支持 RETURNING 子句
