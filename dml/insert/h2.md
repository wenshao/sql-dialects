# H2: INSERT

> 参考资料:
> - [H2 SQL Reference - Commands](https://h2database.com/html/commands.html)
> - [H2 - Data Types](https://h2database.com/html/datatypes.html)
> - [H2 - Functions](https://h2database.com/html/functions.html)
> - 单行插入

```sql
INSERT INTO users (username, email, age) VALUES ('alice', 'alice@example.com', 25);
```

## 多行插入

```sql
INSERT INTO users (username, email, age) VALUES
    ('alice', 'alice@example.com', 25),
    ('bob', 'bob@example.com', 30),
    ('charlie', 'charlie@example.com', 35);
```

## 自增列省略

```sql
INSERT INTO users (username, email, age) VALUES ('alice', 'alice@example.com', 25);
```

## 从查询结果插入

```sql
INSERT INTO users_archive (username, email, age)
SELECT username, email, age FROM users WHERE age > 60;
```

## DEFAULT 值

```sql
INSERT INTO users (username, email) VALUES ('alice', 'alice@example.com');
```

## NULL 值

```sql
INSERT INTO users (username, email, age) VALUES ('alice', 'alice@example.com', NULL);
```

## 带 DEFAULT 关键字

```sql
INSERT INTO events (event_type, status) VALUES ('login', DEFAULT);
```

## CTE + INSERT

```sql
WITH new_users AS (
    SELECT 'dave' AS username, 'dave@example.com' AS email, 28 AS age
    UNION ALL
    SELECT 'eve', 'eve@example.com', 32
)
INSERT INTO users (username, email, age)
SELECT * FROM new_users;
```

## MERGE（H2 特有的 UPSERT 方式）


```sql
MERGE INTO users (id, username, email) KEY (id)
VALUES (1, 'alice', 'alice@example.com');
```

## CSVREAD 批量导入


```sql
INSERT INTO users (username, email, age)
SELECT * FROM CSVREAD('/path/to/users.csv', 'USERNAME,EMAIL,AGE');
```

## 直接建表并导入

```sql
CREATE TABLE imported AS SELECT * FROM CSVREAD('/path/to/data.csv');
```

## CSVWRITE 导出


```sql
CALL CSVWRITE('/path/to/export.csv', 'SELECT * FROM users');
```

## SCRIPT 导出（SQL 格式）


```sql
SCRIPT TO '/path/to/backup.sql';
```

注意：AUTO_INCREMENT / IDENTITY 列在 INSERT 时自动生成
注意：MERGE INTO ... KEY 是 H2 特有的简化 UPSERT
注意：CSVREAD/CSVWRITE 是 H2 内置的 CSV 导入导出功能
注意：H2 兼容多种数据库的 INSERT 语法
