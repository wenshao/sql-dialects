# Larry Ellison - Oracle 创始人

Larry Ellison 是 Oracle Corporation 的联合创始人，
也是将 SQL 从学术研究推向商业市场的关键人物。
Oracle 是第一个成功商业化的 SQL 关系数据库。

## 早期背景

Lawrence Joseph Ellison 出生于 1944 年，在纽约长大。
他在芝加哥大学和伊利诺伊大学学习，但没有完成学位。
1960 年代末他搬到加州，从事计算机编程工作。

在创立 Oracle 之前，Ellison 曾在 Ampex 公司工作，
参与了一个为 CIA 开发的数据库项目，项目代号为"Oracle"——
这个名字后来成为了公司的名称。

## 抓住 System R 的机遇

### 关键论文

1970 年代，IBM 的 System R 项目是 SQL 和关系数据库的先驱实现。
IBM 在学术期刊上公开发表了 System R 的技术论文，
详细描述了 SQL 语言的设计和关系数据库的实现方法。

Ellison 和他的合伙人读到了这些论文，
敏锐地意识到关系数据库有巨大的商业潜力。
而 IBM 当时并不急于将 System R 商业化——
因为 IBM 还在销售自己的层次数据库产品 IMS。

### 抢先商业化

Ellison 决定抢在 IBM 之前，基于公开发表的论文
实现一个商业化的 SQL 关系数据库。

1977 年，Ellison 与 Bob Miner 和 Ed Oates 共同创立了
Software Development Laboratories（SDL），即 Oracle 的前身。

## Oracle 的诞生

### Oracle V2 (1979)

1979 年，公司发布了 Oracle V2——第一个商业化的 SQL 关系数据库。
（之所以叫 V2 而不是 V1，据说是因为 Ellison 认为
客户不愿意购买 1.0 版本的软件。）

Oracle V2 是一个里程碑式的产品：
- 第一个可购买的 SQL 数据库
- 实现了 SQL 语言的核心功能
- 面向商业用户而非学术用途

### 早期发展

Oracle 在 1980 年代迅速发展：
- Oracle V3 (1983)：用 C 语言重写，支持多种操作系统
- Oracle V4 (1984)：引入读一致性
- Oracle V5 (1985)：支持客户端-服务器架构
- Oracle V6 (1988)：引入行级锁、PL/SQL 语言
- Oracle V7 (1992)：引入存储过程、触发器、引用完整性

## Oracle 公司的发展

### 上市与增长

Oracle 于 1986 年上市。在 Ellison 的领导下，
公司通过激进的销售策略和持续的产品创新实现了快速增长。

### 收购策略

2000 年代以后，Oracle 进行了一系列重大收购：
- **PeopleSoft** (2005, 约 103 亿美元)
- **BEA Systems** (2008, 约 85 亿美元)
- **Sun Microsystems** (2010, 约 74 亿美元)——包括 Java 和 MySQL
- **NetSuite** (2016, 约 93 亿美元)

Sun 的收购使得 Oracle 同时拥有了 Java 编程语言和 MySQL 数据库，
这在开源社区引发了广泛的讨论和担忧。

## Oracle SQL 方言

在 Ellison 领导下，Oracle 数据库发展出了独特的 SQL 方言：

### PL/SQL
PL/SQL（Procedural Language/SQL）是 Oracle 的过程化编程语言扩展，
提供了存储过程、函数、包、触发器等功能。
PL/SQL 成为了 Oracle 生态的核心编程语言。

### Oracle 特有语法
Oracle SQL 有许多独特的语法和功能：
- `CONNECT BY` 层次查询
- `DECODE` 函数
- `(+)` 外连接语法（旧式）
- `ROWNUM` 伪列
- `MERGE` 语句（Oracle 率先实现）
- 多版本读一致性

### 标准引领
Oracle 也是多个 SQL 标准功能的早期实现者：
- 窗口函数（分析函数）
- 物化视图
- 闪回查询（Flashback Query）

## 技术领导角色

Ellison 目前担任 Oracle 的 CTO 和执行主席。
近年来，他推动了 Oracle 在以下方向的发展：
- Oracle Cloud Infrastructure (OCI)
- Oracle Autonomous Database（自治数据库）
- 将 Oracle 数据库与云服务深度整合

## 争议与评价

Ellison 是科技行业最具争议性的人物之一。
支持者认为他将 SQL 数据库推向了商业主流，
推动了整个企业软件行业的发展。
批评者则对 Oracle 的激进商业策略和许可政策有意见。

## 影响

无论如何评价 Ellison 的商业手段，
他对数据库行业的影响是不可否认的：

- **商业化 SQL**：Oracle 是第一个证明 SQL 数据库可以商业成功的产品
- **推动行业标准**：Oracle 的市场地位推动了 SQL 成为行业标准
- **企业级功能**：Oracle 数据库推动了许多企业级功能的发展
  （高可用、灾备、安全、性能优化等）
- **生态系统**：Oracle 围绕数据库建立了完整的企业软件生态

从读到 IBM 的学术论文到建立全球最大的数据库公司，
Ellison 的故事是商业敏锐度与技术机遇结合的典型案例。
