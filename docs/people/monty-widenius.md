# Monty Widenius - MySQL 和 MariaDB 创始人

---

Michael "Monty" Widenius 是 MySQL 的主要创始人和首席架构师，
也是 MariaDB 分叉的发起者。他是开源数据库运动中最重要的人物之一。

> 来源：[Wikipedia: Michael Widenius](https://en.wikipedia.org/wiki/Michael_Widenius)

## 早期背景

Monty Widenius 来自芬兰，从年轻时就开始编写数据库相关软件。
在创建 MySQL 之前，他开发了 ISAM 存储引擎和相关工具，
这些早期工作为 MySQL 的存储层奠定了基础。

## MySQL 的诞生 (1995)

1995 年，Widenius 与 David Axmark 和 Allan Larsson 共同创立了 MySQL AB。
MySQL 这个名字来源于 Widenius 的大女儿 My 的名字
（后来 MariaDB 则以他的小女儿 Maria 命名）。
MySQL 首个内部版本发布于 1995 年 5 月 23 日。

> 来源：[Wikipedia: MySQL](https://en.wikipedia.org/wiki/MySQL)、[Wikipedia: Michael Widenius](https://en.wikipedia.org/wiki/Michael_Widenius)

MySQL 的设计目标是：
- 速度快，尤其是读取操作
- 易于安装和使用
- 支持多种操作系统
- 开源（GPL 许可证）

MySQL 迅速成为 LAMP（Linux、Apache、MySQL、PHP）技术栈的核心组件，
在 Web 开发领域获得了巨大的市场份额。

## MySQL AB 时期

MySQL AB 采用了双重许可模式：
- 开源版本使用 GPL 许可证
- 商业版本提供给不希望受 GPL 约束的用户

这种商业模式在当时是开源软件公司的创新之一。

Widenius 在 MySQL AB 期间一直担任 CTO，
亲自编写了 MySQL 的大量核心代码，
特别是存储引擎层和查询处理相关的部分。

## Sun 和 Oracle 收购

2008 年 1 月，Sun Microsystems 以约 10 亿美元收购了 MySQL AB。
2009 年 4 月，Oracle 宣布以 74 亿美元收购 Sun，这意味着 MySQL 将归 Oracle 所有。

> 来源：[Wikipedia: MySQL](https://en.wikipedia.org/wiki/MySQL)

Widenius 对 Oracle 收购 MySQL 表示了强烈的担忧。
他曾公开呼吁欧盟委员会阻止这一收购，
认为 Oracle 作为商业数据库的最大厂商，
不会真心投入 MySQL 的开源发展。

## MariaDB 的创建

2009 年，Widenius 创建了 MariaDB 作为 MySQL 的社区驱动分叉（首个 GA 版本为 MariaDB 5.1，2010 年 2 月发布）。

> 来源：[Wikipedia: MariaDB](https://en.wikipedia.org/wiki/MariaDB)
MariaDB 的目标是：
- 保持与 MySQL 的高度兼容性
- 确保代码始终以 GPL 许可证开源
- 由社区驱动开发方向

MariaDB 逐渐加入了自己的特性，如：
- Aria 存储引擎（取代 MyISAM）
- 增强的查询优化器
- ColumnStore 引擎（列式存储）
- Spider 引擎（分布式分片）

多个主要 Linux 发行版（包括 Red Hat、Debian、SUSE）
将默认的 MySQL 替换为 MariaDB。

## 技术贡献

Widenius 亲自贡献了 MySQL/MariaDB 的大量代码：

- **MyISAM 存储引擎**：MySQL 早期的默认存储引擎
- **Aria 存储引擎**：MyISAM 的崩溃安全替代品
- **查询优化器改进**：MariaDB 中的多项优化器增强
- **复制系统**：MySQL 早期复制功能的实现

## 开源理念

Widenius 是开源运动的坚定支持者。他多次在公开场合表达以下观点：
- 数据库软件应该对所有人开放
- 社区驱动的开发比企业驱动更可持续
- GPL 许可证是保护开源项目的最佳方式

## 公开信息

- **GitHub**: [github.com/montywi](https://github.com/montywi)
- **MariaDB Foundation**: 创始成员和核心开发者
- 经常在开源和数据库相关的会议上演讲

## 影响

MySQL/MariaDB 是 Web 时代最重要的数据库之一。
Wikipedia、Facebook（早期）、Google（部分服务）等都使用过 MySQL。
Widenius 通过创建 MySQL 和 MariaDB，
让关系数据库从昂贵的企业软件变成了人人可用的开源工具。
