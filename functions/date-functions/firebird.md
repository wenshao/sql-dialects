# Firebird: Date Functions

> 参考资料:
> - [Firebird SQL Reference](https://firebirdsql.org/en/reference-manuals/)
> - [Firebird Release Notes](https://firebirdsql.org/file/documentation/release_notes/html/en/4_0/rlsnotes40.html)
> - Current date/time

```sql
SELECT CURRENT_DATE FROM RDB$DATABASE;
SELECT CURRENT_TIME FROM RDB$DATABASE;
SELECT CURRENT_TIMESTAMP FROM RDB$DATABASE;
SELECT 'NOW' FROM RDB$DATABASE;                         -- returns TIMESTAMP
SELECT LOCALTIME FROM RDB$DATABASE;                      -- TIME without time zone
SELECT LOCALTIMESTAMP FROM RDB$DATABASE;                  -- TIMESTAMP without time zone
```

## Date construction

```sql
SELECT DATE '2024-01-15' FROM RDB$DATABASE;
SELECT TIMESTAMP '2024-01-15 10:30:00' FROM RDB$DATABASE;
SELECT CAST('2024-01-15' AS DATE) FROM RDB$DATABASE;
```

## Date arithmetic

```sql
SELECT CURRENT_DATE + 7 FROM RDB$DATABASE;               -- add 7 days
SELECT CURRENT_DATE - 30 FROM RDB$DATABASE;              -- subtract 30 days
SELECT DATEADD(DAY, 7, CURRENT_DATE) FROM RDB$DATABASE;
SELECT DATEADD(MONTH, 3, CURRENT_DATE) FROM RDB$DATABASE;
SELECT DATEADD(YEAR, 1, CURRENT_DATE) FROM RDB$DATABASE;
SELECT DATEADD(HOUR, 2, CURRENT_TIMESTAMP) FROM RDB$DATABASE;
SELECT DATEADD(MINUTE, 30, CURRENT_TIMESTAMP) FROM RDB$DATABASE;
SELECT DATEADD(SECOND, 3600, CURRENT_TIMESTAMP) FROM RDB$DATABASE;
SELECT DATEADD(WEEK, 2, CURRENT_DATE) FROM RDB$DATABASE;
SELECT DATEADD(MILLISECOND, 500, CURRENT_TIMESTAMP) FROM RDB$DATABASE;
```

## Date difference

```sql
SELECT DATEDIFF(DAY, DATE '2024-01-01', DATE '2024-12-31') FROM RDB$DATABASE;    -- 365
SELECT DATEDIFF(MONTH, DATE '2024-01-01', DATE '2024-06-01') FROM RDB$DATABASE;  -- 5
SELECT DATEDIFF(YEAR, DATE '2020-01-01', DATE '2024-01-01') FROM RDB$DATABASE;   -- 4
SELECT DATEDIFF(HOUR, TIMESTAMP '2024-01-01 00:00:00', TIMESTAMP '2024-01-01 12:00:00') FROM RDB$DATABASE; -- 12
SELECT DATEDIFF(WEEK, DATE '2024-01-01', DATE '2024-03-01') FROM RDB$DATABASE;
SELECT DATEDIFF(MILLISECOND, TIMESTAMP '2024-01-01 00:00:00', TIMESTAMP '2024-01-01 00:00:01') FROM RDB$DATABASE;
```

## Extract

```sql
SELECT EXTRACT(YEAR FROM CURRENT_DATE) FROM RDB$DATABASE;
SELECT EXTRACT(MONTH FROM CURRENT_DATE) FROM RDB$DATABASE;
SELECT EXTRACT(DAY FROM CURRENT_DATE) FROM RDB$DATABASE;
SELECT EXTRACT(HOUR FROM CURRENT_TIMESTAMP) FROM RDB$DATABASE;
SELECT EXTRACT(MINUTE FROM CURRENT_TIMESTAMP) FROM RDB$DATABASE;
SELECT EXTRACT(SECOND FROM CURRENT_TIMESTAMP) FROM RDB$DATABASE;
SELECT EXTRACT(WEEKDAY FROM CURRENT_DATE) FROM RDB$DATABASE;     -- 0=Sunday
SELECT EXTRACT(YEARDAY FROM CURRENT_DATE) FROM RDB$DATABASE;     -- day of year
SELECT EXTRACT(WEEK FROM CURRENT_DATE) FROM RDB$DATABASE;        -- ISO week (4.0+)
SELECT EXTRACT(MILLISECOND FROM CURRENT_TIMESTAMP) FROM RDB$DATABASE;
```

## 4.0+: time zone functions

```sql
SELECT CURRENT_TIMESTAMP AT TIME ZONE 'America/New_York' FROM RDB$DATABASE;
SELECT CURRENT_TIMESTAMP AT LOCAL FROM RDB$DATABASE;
SELECT EXTRACT(TIMEZONE_HOUR FROM CURRENT_TIME) FROM RDB$DATABASE;
SELECT EXTRACT(TIMEZONE_MINUTE FROM CURRENT_TIME) FROM RDB$DATABASE;
```

## Date formatting (no built-in TO_CHAR; use EXTRACT and concatenation)

```sql
SELECT EXTRACT(YEAR FROM CURRENT_DATE) || '-' ||
       LPAD(EXTRACT(MONTH FROM CURRENT_DATE), 2, '0') || '-' ||
       LPAD(EXTRACT(DAY FROM CURRENT_DATE), 2, '0')
FROM RDB$DATABASE;
```

Note: DATEADD and DATEDIFF are Firebird's primary date arithmetic functions
Note: date +/- integer adds/subtracts days directly
Note: RDB$DATABASE is the single-row system table
Note: no TO_CHAR or FORMAT function; build strings manually
Note: 4.0+ adds time zone support and EXTRACT(WEEK)
