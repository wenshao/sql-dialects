# H2: 日期函数

## 当前时间

```sql
SELECT CURRENT_DATE;
SELECT CURRENT_TIME;
SELECT CURRENT_TIMESTAMP;
SELECT NOW();
SELECT LOCALTIMESTAMP;
```

## 日期加减

```sql
SELECT DATEADD('DAY', 7, CURRENT_DATE);
SELECT DATEADD('MONTH', 1, CURRENT_DATE);
SELECT DATEADD('HOUR', 3, CURRENT_TIMESTAMP);
SELECT DATEADD('YEAR', -1, CURRENT_DATE);
```

## 日期差

```sql
SELECT DATEDIFF('DAY', DATE '2024-01-01', DATE '2024-12-31');
SELECT DATEDIFF('MONTH', DATE '2024-01-01', DATE '2024-06-15');
SELECT DATEDIFF('HOUR', TIMESTAMP '2024-01-01 00:00:00', TIMESTAMP '2024-01-02 12:00:00');
```

## 提取

```sql
SELECT EXTRACT(YEAR FROM CURRENT_DATE);
SELECT EXTRACT(MONTH FROM CURRENT_DATE);
SELECT EXTRACT(DAY FROM CURRENT_DATE);
SELECT YEAR(CURRENT_DATE);
SELECT MONTH(CURRENT_DATE);
SELECT DAY(CURRENT_DATE);
SELECT HOUR(CURRENT_TIMESTAMP);
SELECT MINUTE(CURRENT_TIMESTAMP);
SELECT SECOND(CURRENT_TIMESTAMP);
SELECT DAY_OF_WEEK(CURRENT_DATE);
SELECT DAY_OF_YEAR(CURRENT_DATE);
SELECT ISO_WEEK(CURRENT_DATE);
SELECT ISO_YEAR(CURRENT_DATE);
```

## 截断

```sql
SELECT DATE_TRUNC('HOUR', CURRENT_TIMESTAMP);
SELECT DATE_TRUNC('DAY', CURRENT_TIMESTAMP);
```

## 格式化

```sql
SELECT FORMATDATETIME(CURRENT_TIMESTAMP, 'yyyy-MM-dd HH:mm:ss');
SELECT FORMATDATETIME(CURRENT_TIMESTAMP, 'EEEE, dd MMMM yyyy');
```

## 解析

```sql
SELECT PARSEDATETIME('2024-01-15 10:30:00', 'yyyy-MM-dd HH:mm:ss');
```

## 构造

```sql
SELECT CURRENT_DATE + 7;                     -- 加天数
```

注意：DATEADD/DATEDIFF 是核心函数
注意：支持 EXTRACT 和快捷函数（YEAR, MONTH 等）
注意：FORMATDATETIME 使用 Java SimpleDateFormat 模式
