# 达梦 (DM): Type Conversion

> 参考资料:
> - [达梦数据库 SQL 参考手册](https://eco.dameng.com/document/dm/zh-cn/sql-dev/)


## CAST (SQL 标准)

```sql
SELECT CAST(42 AS VARCHAR2(10)) FROM DUAL;
SELECT CAST('42' AS NUMBER) FROM DUAL;
SELECT CAST('2024-01-15' AS DATE) FROM DUAL;
```

## TO_CHAR / TO_NUMBER / TO_DATE (Oracle 兼容)

```sql
SELECT TO_CHAR(123456.789, '999,999.99') FROM DUAL;
SELECT TO_CHAR(SYSDATE, 'YYYY-MM-DD HH24:MI:SS') FROM DUAL;
SELECT TO_NUMBER('123,456.78', '999,999.99') FROM DUAL;
SELECT TO_DATE('2024-01-15', 'YYYY-MM-DD') FROM DUAL;
SELECT TO_TIMESTAMP('2024-01-15 10:30:00', 'YYYY-MM-DD HH24:MI:SS') FROM DUAL;
```

## 隐式转换 (Oracle 兼容)

```sql
SELECT '42' + 0 FROM DUAL;
```

## 更多数值转换

```sql
SELECT CAST(3.14 AS INTEGER) FROM DUAL;              -- 3 (截断)
SELECT CAST('100' AS NUMBER(10)) FROM DUAL;          -- 100
SELECT CAST(3.14 AS NUMBER(10,1)) FROM DUAL;         -- 3.1
```

## 日期/时间格式化

```sql
SELECT TO_CHAR(SYSDATE, 'YYYY-MM-DD HH24:MI:SS') FROM DUAL;
SELECT TO_CHAR(SYSDATE, 'Day, DD Month YYYY') FROM DUAL;
SELECT TO_CHAR(SYSDATE, 'YYYY"年"MM"月"DD"日"') FROM DUAL;
SELECT TO_DATE('15/01/2024', 'DD/MM/YYYY') FROM DUAL;
SELECT TO_DATE('Jan 15, 2024', 'Mon DD, YYYY') FROM DUAL;
SELECT TO_TIMESTAMP('2024-01-15 10:30:00.123', 'YYYY-MM-DD HH24:MI:SS.FF3') FROM DUAL;
```

## 数值格式化

```sql
SELECT TO_CHAR(1234567.89, 'FM9,999,999.00') FROM DUAL;  -- '1,234,567.89'
SELECT TO_CHAR(0.15, 'FM990.00%') FROM DUAL;
SELECT TO_NUMBER('$1,234.56', 'L9,999.99') FROM DUAL;
```

## NVL / DECODE 转换

```sql
SELECT NVL(NULL, '默认值') FROM DUAL;
SELECT DECODE(1, 1, '一', 2, '二', '其他') FROM DUAL;
```

## 日期部分提取

```sql
SELECT EXTRACT(YEAR FROM SYSDATE) FROM DUAL;
SELECT EXTRACT(MONTH FROM SYSDATE) FROM DUAL;
SELECT EXTRACT(DAY FROM SYSDATE) FROM DUAL;
```

## 区间转换

```sql
SELECT SYSDATE + INTERVAL '1' DAY FROM DUAL;
SELECT SYSDATE + INTERVAL '2' HOUR FROM DUAL;
SELECT SYSDATE - INTERVAL '30' MINUTE FROM DUAL;
```

## CLOB / BLOB 转换

```sql
SELECT CAST('long text...' AS CLOB) FROM DUAL;
```

## 隐式转换 (Oracle 兼容)

```sql
SELECT '42' + 0 FROM DUAL;                           -- 42
SELECT 1 + 1.5 FROM DUAL;                           -- 2.5
```

错误处理（无 TRY_CAST）
转换失败直接报错
使用 PL/SQL EXCEPTION 块封装安全转换
注意：达梦兼容 Oracle 类型转换函数
注意：日期格式使用 Oracle 模式 (YYYY, MM, DD, HH24, MI, SS)
注意：需要使用 FROM DUAL
限制：无 TRY_CAST, ::, CONVERT (SQL Server 风格)
