# Firebird: Type Conversion

> 参考资料:
> - [Firebird Language Reference - CAST](https://firebirdsql.org/file/documentation/html/en/refdocs/fblangref40/)

```sql
SELECT CAST(42 AS VARCHAR(10)); SELECT CAST('42' AS INTEGER);
SELECT CAST('3.14' AS DECIMAL(10,2)); SELECT CAST('2024-01-15' AS DATE);
SELECT CAST('10:30:00' AS TIME); SELECT CAST('2024-01-15 10:30:00' AS TIMESTAMP);
```

格式化 (无内置格式化函数，需在应用层处理)
使用 LPAD/SUBSTRING 等手动格式化
隐式转换

```sql
SELECT '42' || 0;                                -- 文本拼接（不自动转数字）
```

## 更多数值转换

```sql
SELECT CAST(3.14 AS INTEGER) FROM RDB$DATABASE;      -- 3 (截断)
SELECT CAST(3.14 AS SMALLINT) FROM RDB$DATABASE;     -- 3
SELECT CAST(3.14 AS BIGINT) FROM RDB$DATABASE;       -- 3
SELECT CAST(42 AS DOUBLE PRECISION) FROM RDB$DATABASE; -- 42.0
SELECT CAST(42 AS NUMERIC(10,2)) FROM RDB$DATABASE;  -- 42.00
```

## 字符串 ↔ 数值

```sql
SELECT CAST(12345 AS VARCHAR(10)) FROM RDB$DATABASE;
SELECT CAST('67890' AS BIGINT) FROM RDB$DATABASE;
```

## 布尔转换 (Firebird 3.0+)

```sql
SELECT CAST(TRUE AS VARCHAR(5)) FROM RDB$DATABASE;   -- 'TRUE'
```

## 日期/时间转换

```sql
SELECT CAST('2024-01-15' AS DATE) FROM RDB$DATABASE;
SELECT CAST('10:30:00' AS TIME) FROM RDB$DATABASE;
SELECT CAST('2024-01-15 10:30:00' AS TIMESTAMP) FROM RDB$DATABASE;
SELECT CAST(CURRENT_DATE AS VARCHAR(10)) FROM RDB$DATABASE;
SELECT CAST(CURRENT_TIMESTAMP AS VARCHAR(24)) FROM RDB$DATABASE;
```

## 日期部分提取 (替代 TO_CHAR)

```sql
SELECT EXTRACT(YEAR FROM CURRENT_DATE) FROM RDB$DATABASE;
SELECT EXTRACT(MONTH FROM CURRENT_DATE) FROM RDB$DATABASE;
SELECT EXTRACT(DAY FROM CURRENT_DATE) FROM RDB$DATABASE;
SELECT EXTRACT(HOUR FROM CURRENT_TIMESTAMP) FROM RDB$DATABASE;
```

## 手动日期格式化 (无内置格式化函数)

```sql
SELECT EXTRACT(YEAR FROM CURRENT_DATE) || '-' ||
       LPAD(EXTRACT(MONTH FROM CURRENT_DATE), 2, '0') || '-' ||
       LPAD(EXTRACT(DAY FROM CURRENT_DATE), 2, '0')
FROM RDB$DATABASE;
```

## 隐式转换

```sql
SELECT '42' || ' items' FROM RDB$DATABASE;           -- 字符串拼接
SELECT 1 + 1.5 FROM RDB$DATABASE;                   -- NUMERIC
```

## 精度处理

```sql
SELECT CAST(1.0/3.0 AS DECIMAL(10,4)) FROM RDB$DATABASE; -- 0.3333
```

错误处理（无 TRY_CAST，失败报错）
建议在应用层用 try-catch 处理
注意：Firebird 主要使用标准 CAST
注意：Firebird 使用 FROM RDB$DATABASE 替代 FROM DUAL
注意：日期格式化需手动拼接或在应用层处理
限制：无 TRY_CAST, CONVERT, ::, TO_NUMBER, TO_CHAR, TO_DATE
限制：无内置日期/数值格式化函数
