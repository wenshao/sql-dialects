# SAP HANA: Type Conversion

> 参考资料:
> - [SAP HANA SQL Reference - Conversion Functions](https://help.sap.com/docs/SAP_HANA_PLATFORM/4fe29514fd584807ac9f2a04f6754767/)

```sql
SELECT CAST(42 AS VARCHAR(10)) FROM DUMMY; SELECT CAST('42' AS INTEGER) FROM DUMMY;
SELECT CAST('3.14' AS DECIMAL(10,2)) FROM DUMMY; SELECT CAST('2024-01-15' AS DATE) FROM DUMMY;
```

## TO_* 转换函数

```sql
SELECT TO_VARCHAR(42) FROM DUMMY;
SELECT TO_INTEGER('42') FROM DUMMY;
SELECT TO_DECIMAL('3.14', 10, 2) FROM DUMMY;
SELECT TO_DATE('2024-01-15', 'YYYY-MM-DD') FROM DUMMY;
SELECT TO_TIMESTAMP('2024-01-15 10:30:00', 'YYYY-MM-DD HH24:MI:SS') FROM DUMMY;
SELECT TO_NVARCHAR(42) FROM DUMMY;
```

## 格式化

```sql
SELECT TO_VARCHAR(CURRENT_DATE, 'YYYY-MM-DD') FROM DUMMY;
SELECT TO_VARCHAR(123456.789, '999,999.99') FROM DUMMY;
```

## 更多数值转换

```sql
SELECT TO_BIGINT('100') FROM DUMMY;                  -- 100
SELECT TO_REAL(42) FROM DUMMY;                       -- 42.0
SELECT TO_DOUBLE('3.14') FROM DUMMY;                 -- 3.14
SELECT TO_SMALLINT(42) FROM DUMMY;                   -- 42
SELECT TO_TINYINT(42) FROM DUMMY;                    -- 42
```

## 布尔转换

```sql
SELECT TO_BOOLEAN(1) FROM DUMMY;                     -- TRUE
SELECT TO_BOOLEAN(0) FROM DUMMY;                     -- FALSE
SELECT TO_BOOLEAN('true') FROM DUMMY;                -- TRUE
SELECT TO_INTEGER(TRUE) FROM DUMMY;                  -- 1
```

## 日期/时间格式化

```sql
SELECT TO_VARCHAR(CURRENT_DATE, 'YYYY-MM-DD') FROM DUMMY;
SELECT TO_VARCHAR(CURRENT_TIMESTAMP, 'YYYY-MM-DD HH24:MI:SS') FROM DUMMY;
SELECT TO_VARCHAR(CURRENT_DATE, 'Day, DD Month YYYY') FROM DUMMY;
SELECT TO_VARCHAR(CURRENT_DATE, 'YYYY"年"MM"月"DD"日"') FROM DUMMY;
SELECT TO_DATE('15/01/2024', 'DD/MM/YYYY') FROM DUMMY;
SELECT TO_TIMESTAMP('2024-01-15 10:30:00.123', 'YYYY-MM-DD HH24:MI:SS.FF3') FROM DUMMY;
```

## 数值格式化

```sql
SELECT TO_VARCHAR(1234567.89, '9,999,999.00') FROM DUMMY;
SELECT TO_VARCHAR(-1234.5, '9,999.00MI') FROM DUMMY;
```

## 日期部分提取

```sql
SELECT YEAR(CURRENT_DATE) FROM DUMMY;
SELECT MONTH(CURRENT_DATE) FROM DUMMY;
SELECT DAYOFMONTH(CURRENT_DATE) FROM DUMMY;
SELECT HOUR(CURRENT_TIMESTAMP) FROM DUMMY;
```

## NVARCHAR / NCLOB 转換

```sql
SELECT TO_NVARCHAR(42) FROM DUMMY;
SELECT TO_NCLOB('long text...') FROM DUMMY;
```

## 二進制转换

```sql
SELECT TO_BINARY('48454C4C4F') FROM DUMMY;           -- HEX → BINARY
SELECT TO_VARCHAR(TO_BINARY('HELLO'), 'UTF-8') FROM DUMMY;
```

## 隐式转換

```sql
SELECT 1 + 1.5 FROM DUMMY;                          -- DECIMAL
```

## 精度処理

```sql
SELECT ROUND(3.14159, 2) FROM DUMMY;                 -- 3.14
SELECT CEIL(3.14) FROM DUMMY;                        -- 4
SELECT FLOOR(3.14) FROM DUMMY;                       -- 3
```

错误処理（无 TRY_CAST）
转换失败直接报错
建议使用 CASE + LIKE/LOCATE 预验证
注意：SAP HANA 使用 TO_* 系列转换函数
注意：需要 FROM DUMMY (类似 Oracle 的 FROM DUAL)
注意：默认字符类型是 NVARCHAR (Unicode)
注意：日期格式使用 Oracle 风格模式
限制：无 TRY_CAST, ::, CONVERT (SQL Server 风格)
