> - LIMIT / OFFSET

```sql
SELECT * FROM users ORDER BY id LIMIT 10 OFFSET 20;
```

## 仅 LIMIT

```sql
SELECT * FROM users ORDER BY id LIMIT 10;
```

## FETCH FIRST（SQL 标准语法）

```sql
SELECT * FROM users ORDER BY id FETCH FIRST 10 ROWS ONLY;
SELECT * FROM users ORDER BY id OFFSET 20 ROWS FETCH NEXT 10 ROWS ONLY;
```

## FETCH ... WITH TIES（包含相同值的行）

```sql
SELECT * FROM users ORDER BY age FETCH FIRST 10 ROWS WITH TIES;
```

## 窗口函数分页

```sql
SELECT * FROM (
    SELECT *, ROW_NUMBER() OVER (ORDER BY id) AS rn
    FROM users
) t
WHERE rn BETWEEN 21 AND 30;
```

## 游标分页（Keyset Pagination）

```sql
SELECT * FROM users WHERE id > 100 ORDER BY id LIMIT 10;
```

## TOP（SQL Server 兼容模式）


## 在 SQL Server 兼容模式下可用

SELECT TOP 10 * FROM users ORDER BY id;

## FETCH FIRST ... PERCENT

```sql
SELECT * FROM users ORDER BY age FETCH FIRST 10 PERCENT ROWS ONLY;
```

## 取每组前 N 条


```sql
SELECT * FROM (
    SELECT *, ROW_NUMBER() OVER (PARTITION BY city ORDER BY created_at DESC) AS rn
    FROM users
) WHERE rn <= 3;
```

注意：H2 支持 LIMIT/OFFSET 和 FETCH FIRST 两种语法
注意：支持 WITH TIES（包含相同排序值的行）
注意：支持 PERCENT（按百分比取行）
注意：游标分页在大数据量下性能更好
注意：在不同兼容模式下可能支持 TOP 语法
