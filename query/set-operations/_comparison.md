## е…ій”®е·®ејӮ

- **Oracle** дј з»ҹдҪҝз”Ё MINUS д»Јжӣҝ EXCEPTпјҢ21c иө·еҗҢж—¶ж”ҜжҢҒдёӨиҖ…
- **MySQL 8.0.31 д№ӢеүҚ** дёҚж”ҜжҢҒ INTERSECT е’Ң EXCEPTпјҲйңҖз”Ё JOIN/NOT EXISTS жЁЎжӢҹпјү
- **SQL Server** дёҚж”ҜжҢҒ INTERSECT ALL е’Ң EXCEPT ALL
- **SQLite** дёҚж”ҜжҢҒ INTERSECT ALL е’Ң EXCEPT ALLпјҢдёҚж”ҜжҢҒжӢ¬еҸ·жҺ§еҲ¶дјҳе…Ҳзә§
- **Hive** дёҚж”ҜжҢҒ EXCEPTпјҲйңҖз”Ё LEFT SEMI JOIN жӣҝд»Јпјү
- **Impala** дёҚж”ҜжҢҒ INTERSECT е’Ң EXCEPT
- **Teradata** дҪҝз”Ё MINUS иҖҢйқһ EXCEPT
- **йӣҶеҗҲж“ҚдҪңзҡ„дјҳе…Ҳзә§**пјҡINTERSECT еңЁеӨ§еӨҡж•°ж•°жҚ®еә“дёӯдјҳе…Ҳзә§й«ҳдәҺ UNION е’Ң EXCEPT
- **UNION DISTINCT** д»… MySQL/MariaDB жҳҫејҸж”ҜжҢҒпјҢе…¶д»–ж•°жҚ®еә“ UNION й»ҳи®ӨеҚідёә DISTINCT
