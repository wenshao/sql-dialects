## 示例数据（H2 2.x 支持 JSON 类型）

```sql
CREATE TABLE orders_json (
    id   INT AUTO_INCREMENT PRIMARY KEY,
    data JSON NOT NULL
);

INSERT INTO orders_json (data) VALUES
('{"customer":"Alice","total":150.0,"items":[{"product":"Widget","qty":2,"price":25.0}],"address":{"city":"Beijing","zip":"100000"}}');
```

## JSON 值提取

```sql
SELECT id,
       CAST(data FORMAT JSON AS VARCHAR) AS raw_json
FROM   orders_json;
```
