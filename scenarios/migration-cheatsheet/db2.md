# DB2: 迁移速查表 (Migration Cheatsheet)

> 参考资料:
> - [IBM DB2 SQL Reference](https://www.ibm.com/docs/en/db2/11.5)
> - [IBM DB2 Migration Guide](https://www.ibm.com/docs/en/db2/11.5?topic=migrating)


一、数据类型映射（从常见数据库到 DB2）
INT→INTEGER, BIGINT→BIGINT, FLOAT→REAL, DOUBLE→DOUBLE,
VARCHAR→VARCHAR, TEXT→CLOB, DECIMAL→DECIMAL(p,s),
BOOLEAN→BOOLEAN(11.1+)或SMALLINT, DATE→DATE, TIMESTAMP→TIMESTAMP,
BLOB→BLOB, JSON→CLOB(用JSON函数), AUTO_INCREMENT→GENERATED ALWAYS AS IDENTITY
二、函数映射: IFNULL/NVL→COALESCE, NOW()→CURRENT TIMESTAMP,
CONCAT→CONCAT或||, GROUP_CONCAT→LISTAGG(11.1+),
DATEDIFF→TIMESTAMPDIFF, DATE_ADD→d+n DAYS
三、陷阱: DB2对标准SQL遵循度高, 表名schema.table必须指定,
日期算术用DAYS/MONTHS/YEARS关键字, 字符串连接用||或CONCAT
四、自增: GENERATED ALWAYS AS IDENTITY 或 SEQUENCE
五、日期: CURRENT TIMESTAMP; CURRENT DATE; d + 1 DAY;
TIMESTAMPDIFF(16, CHAR(t2-t1)); VARCHAR_FORMAT(ts,'YYYY-MM-DD')
TO_DATE(s,'YYYY-MM-DD'); TO_CHAR(d,'YYYY-MM-DD'); -- 11.1+
YEAR(d); MONTH(d); DAY(d); HOUR(ts)
六、字符串: LENGTH, UPPER, LOWER, TRIM, SUBSTR, REPLACE, LOCATE, ||, LISTAGG

## 七、数据类型映射（从 Oracle/MySQL/PostgreSQL 到 DB2）

Oracle → DB2:
NUMBER(p,s) → DECIMAL(p,s), VARCHAR2(n) → VARCHAR(n),
CLOB → CLOB, DATE → TIMESTAMP (Oracle DATE含时间),
TIMESTAMP → TIMESTAMP, BLOB → BLOB,
SYSDATE → CURRENT TIMESTAMP, SEQUENCE → SEQUENCE,
ROWID → 不支持
MySQL → DB2:
INT → INTEGER, BIGINT → BIGINT, FLOAT → REAL,
DOUBLE → DOUBLE, VARCHAR(n) → VARCHAR(n),
TEXT → CLOB, DATETIME → TIMESTAMP, DATE → DATE,
DECIMAL(p,s) → DECIMAL(p,s),
BOOLEAN → BOOLEAN (11.1+) 或 SMALLINT,
AUTO_INCREMENT → GENERATED ALWAYS AS IDENTITY,
JSON → CLOB (用JSON函数)
PostgreSQL → DB2:
INTEGER → INTEGER, TEXT → CLOB, SERIAL → IDENTITY,
BOOLEAN → BOOLEAN (11.1+), JSONB → CLOB,
BYTEA → BLOB, ARRAY → 不支持
八、函数等价映射
MySQL → DB2:
IFNULL → COALESCE, NOW() → CURRENT TIMESTAMP,
DATE_FORMAT → VARCHAR_FORMAT, CONCAT(a,b) → a || b,
GROUP_CONCAT → LISTAGG (11.1+),
LIMIT → FETCH FIRST n ROWS ONLY, DATE_ADD → d + n DAYS
九、常见陷阱补充
标准 SQL 遵循度高
表名格式: schema.table（必须指定 schema）
日期算术用 DAYS/MONTHS/YEARS 关键字
CHAR() 函数的 ISO/USA/EUR/JIS 格式是 DB2 特有
RUNSTATS 替代 ANALYZE
REORG TABLE 替代 VACUUM
MQT (物化查询表) 替代物化视图
十、NULL 处理
COALESCE(a, b, c); NULLIF(a, b);
VALUE(a, b);                                       -- DB2 特有，同 COALESCE
十一、分页语法
SELECT * FROM t ORDER BY id FETCH FIRST 10 ROWS ONLY;
SELECT * FROM t ORDER BY id OFFSET 20 ROWS FETCH NEXT 10 ROWS ONLY; -- 11.1+
十二、日期格式码 (类 Oracle 风格)
YYYY=年, MM=月, DD=日, HH24=24时, MI=分, SS=秒
