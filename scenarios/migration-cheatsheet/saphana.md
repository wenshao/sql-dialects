# SAP HANA: 迁移速查表 (Migration Cheatsheet)

> 参考资料:
> - [SAP HANA SQL Reference](https://help.sap.com/docs/HANA_CLOUD)


一、数据类型（到 SAP HANA）
INT→INTEGER, BIGINT→BIGINT, FLOAT→REAL, DOUBLE→DOUBLE,
VARCHAR→NVARCHAR(n)(Unicode默认), TEXT→NCLOB, DECIMAL→DECIMAL(p,s),
BOOLEAN→BOOLEAN, DATE→DATE, TIMESTAMP→TIMESTAMP,
BLOB→BLOB, JSON→NCLOB(用JSON函数), AUTO_INCREMENT→GENERATED ALWAYS AS IDENTITY
二、函数: IFNULL→IFNULL/COALESCE, NOW()→CURRENT_TIMESTAMP,
CONCAT→||或CONCAT, GROUP_CONCAT→STRING_AGG
三、陷阱: 列存储为默认(行存储需显式指定), 内存数据库(数据量受内存限制),
NVARCHAR默认(Unicode), 计算视图/分析视图是HANA特色, 多租户架构
四、自增: GENERATED ALWAYS AS IDENTITY 或 SEQUENCE
五、日期: CURRENT_TIMESTAMP; CURRENT_DATE; ADD_DAYS(d,1);
DAYS_BETWEEN(a,b); TO_VARCHAR(ts,'YYYY-MM-DD HH24:MI:SS')
ADD_MONTHS(d,1); ADD_YEARS(d,1); ADD_SECONDS(ts,3600)
EXTRACT(YEAR FROM d); WEEKDAY(d); LAST_DAY(d)
六、字符串: LENGTH, UPPER, LOWER, TRIM, SUBSTR, REPLACE, LOCATE, ||, STRING_AGG

## 七、数据类型映射（从 Oracle/MySQL/PostgreSQL 到 SAP HANA）

Oracle → SAP HANA:
NUMBER(p,s) → DECIMAL(p,s), VARCHAR2(n) → NVARCHAR(n),
CLOB → NCLOB, DATE → DATE (注意: Oracle DATE含时间, HANA不含),
TIMESTAMP → TIMESTAMP, BLOB → BLOB,
SYSDATE → CURRENT_TIMESTAMP, SEQUENCE → SEQUENCE
MySQL → SAP HANA:
INT → INTEGER, BIGINT → BIGINT, FLOAT → REAL,
DOUBLE → DOUBLE, VARCHAR(n) → NVARCHAR(n),
TEXT → NCLOB, DATETIME → TIMESTAMP, DATE → DATE,
DECIMAL(p,s) → DECIMAL(p,s), BOOLEAN → BOOLEAN,
AUTO_INCREMENT → GENERATED ALWAYS AS IDENTITY,
JSON → NCLOB (用JSON函数)
PostgreSQL → SAP HANA:
INTEGER → INTEGER, TEXT → NCLOB, SERIAL → IDENTITY,
BOOLEAN → BOOLEAN, JSONB → NCLOB, BYTEA → BLOB
八、函数等价映射
MySQL → SAP HANA:
IFNULL → IFNULL/COALESCE, NOW() → CURRENT_TIMESTAMP,
DATE_FORMAT → TO_VARCHAR, CONCAT(a,b) → a || b,
GROUP_CONCAT → STRING_AGG, LIMIT → LIMIT (HANA 支持)
Oracle → SAP HANA:
NVL → IFNULL, SYSDATE → CURRENT_TIMESTAMP,
TO_CHAR → TO_VARCHAR, DECODE → MAP 或 CASE WHEN
九、常见陷阱补充
列存储为默认（行存储需 COLUMN/ROW TABLE 指定）
内存数据库（数据量受内存限制）
NVARCHAR 默认（Unicode），VARCHAR 是 ASCII
计算视图/分析视图是 HANA 特色
多租户架构（System/Tenant DB）
从 DUMMY 表取常量（类似 Oracle 的 DUAL）
十、NULL 处理
IFNULL(a, b); COALESCE(a, b, c);
NULLIF(a, b); MAP(a, NULL, b, a) -- 类似 NVL
十一、分页语法
SELECT * FROM t ORDER BY id LIMIT 10 OFFSET 20;
十二、日期格式码 (Oracle 风格)
YYYY=年, MM=月, DD=日, HH24=24时, MI=分, SS=秒
