# Derby: 日期时间类型

> 参考资料:
> - [Derby SQL Reference](https://db.apache.org/derby/docs/10.16/ref/)
> - [Derby Developer Guide](https://db.apache.org/derby/docs/10.16/devguide/)


DATE: 日期（yyyy-mm-dd）
TIME: 时间（hh:mm:ss）
TIMESTAMP: 日期时间（yyyy-mm-dd hh:mm:ss.fffffffff）

```sql
CREATE TABLE events (
    id         INT NOT NULL GENERATED ALWAYS AS IDENTITY,
    event_date DATE,
    event_time TIME,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    PRIMARY KEY (id)
);
```

## 获取当前时间

```sql
SELECT CURRENT_DATE FROM SYSIBM.SYSDUMMY1;
SELECT CURRENT_TIME FROM SYSIBM.SYSDUMMY1;
SELECT CURRENT_TIMESTAMP FROM SYSIBM.SYSDUMMY1;
```

## VALUES 语法（Derby 特有）

```sql
VALUES CURRENT_DATE;
VALUES CURRENT_TIME;
VALUES CURRENT_TIMESTAMP;
```

## 日期字面量

```sql
VALUES DATE('2024-01-15');
VALUES TIME('10:30:00');
VALUES TIMESTAMP('2024-01-15 10:30:00');
```

日期运算（使用函数或表达式）
Derby 不支持直接的 + INTERVAL 语法
使用 {fn TIMESTAMPADD(...)} JDBC 转义语法

```sql
SELECT {fn TIMESTAMPADD(SQL_TSI_DAY, 7, CURRENT_DATE)} FROM SYSIBM.SYSDUMMY1;
SELECT {fn TIMESTAMPADD(SQL_TSI_HOUR, 3, CURRENT_TIMESTAMP)} FROM SYSIBM.SYSDUMMY1;
SELECT {fn TIMESTAMPADD(SQL_TSI_MONTH, 1, CURRENT_DATE)} FROM SYSIBM.SYSDUMMY1;
```

## 日期差

```sql
SELECT {fn TIMESTAMPDIFF(SQL_TSI_DAY, DATE('2024-01-01'), DATE('2024-12-31'))}
FROM SYSIBM.SYSDUMMY1;
```

## 提取

```sql
SELECT YEAR(CURRENT_DATE) FROM SYSIBM.SYSDUMMY1;
SELECT MONTH(CURRENT_DATE) FROM SYSIBM.SYSDUMMY1;
SELECT DAY(CURRENT_DATE) FROM SYSIBM.SYSDUMMY1;
SELECT HOUR(CURRENT_TIMESTAMP) FROM SYSIBM.SYSDUMMY1;
SELECT MINUTE(CURRENT_TIMESTAMP) FROM SYSIBM.SYSDUMMY1;
SELECT SECOND(CURRENT_TIMESTAMP) FROM SYSIBM.SYSDUMMY1;
```

## 类型转换

```sql
SELECT CAST('2024-01-15' AS DATE) FROM SYSIBM.SYSDUMMY1;
SELECT CAST('10:30:00' AS TIME) FROM SYSIBM.SYSDUMMY1;
SELECT CAST('2024-01-15 10:30:00' AS TIMESTAMP) FROM SYSIBM.SYSDUMMY1;
```

## VARCHAR 转换

```sql
SELECT CHAR(CURRENT_DATE) FROM SYSIBM.SYSDUMMY1;
SELECT VARCHAR(CURRENT_TIMESTAMP) FROM SYSIBM.SYSDUMMY1;
```

注意：不支持 TIMESTAMP WITH TIME ZONE
注意：不支持 INTERVAL 类型
注意：日期运算使用 JDBC 转义函数
注意：SYSIBM.SYSDUMMY1 是单行虚拟表
注意：TIMESTAMP 精度为纳秒（9位）
