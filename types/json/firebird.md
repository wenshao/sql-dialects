# Firebird: JSON Support

> 参考资料:
> - [Firebird SQL Reference](https://firebirdsql.org/en/reference-manuals/)
> - [Firebird Release Notes](https://firebirdsql.org/file/documentation/release_notes/html/en/4_0/rlsnotes40.html)


Firebird does not have a native JSON type
JSON is stored as VARCHAR or BLOB SUB_TYPE TEXT
JSON processing available via built-in functions (5.0+) or UDFs

```sql
CREATE TABLE events (
    id   BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    data VARCHAR(8000)               -- store JSON as text
);
```

## For larger JSON documents

```sql
CREATE TABLE documents (
    id   BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
    data BLOB SUB_TYPE TEXT CHARACTER SET UTF8
);
```

## Insert JSON

```sql
INSERT INTO events (data) VALUES ('{"name": "alice", "age": 25, "tags": ["vip", "new"]}');
```

Prior to 5.0: no built-in JSON functions
Use string functions for basic extraction
Or use external UDF libraries
Manual JSON value extraction (limited, for simple cases)

```sql
SELECT
    SUBSTRING(data FROM POSITION('"name"' IN data) + 8
              FOR POSITION('"' IN SUBSTRING(data FROM POSITION('"name"' IN data) + 8)) - 1)
FROM events;
```

## CONTAINING for JSON search (case-insensitive substring)

```sql
SELECT * FROM events WHERE data CONTAINING '"alice"';
SELECT * FROM events WHERE data CONTAINING '"vip"';
```

## LIKE for JSON pattern matching

```sql
SELECT * FROM events WHERE data LIKE '%"name": "alice"%';
```

## Domain for JSON validation

```sql
CREATE DOMAIN d_json AS VARCHAR(8000)
    CHECK (VALUE IS NULL OR VALUE CONTAINING '{' OR VALUE CONTAINING '[');
```

## EXECUTE BLOCK for JSON processing

```sql
SET TERM !! ;
EXECUTE BLOCK
RETURNS (name VARCHAR(100), age INTEGER)
AS
    DECLARE json_data VARCHAR(8000);
    DECLARE pos INTEGER;
BEGIN
    FOR SELECT data FROM events INTO :json_data DO
    BEGIN
        SUSPEND;
    END
END!!
SET TERM ; !!
```

External UDF approach (using ib_udf or custom library)
DECLARE EXTERNAL FUNCTION json_extract
CSTRING(8000), CSTRING(200)
RETURNS CSTRING(8000)
ENTRY_POINT 'json_extract'
MODULE_NAME 'json_udf';
Note: Firebird has no native JSON type or functions (pre-5.0)
Note: JSON is stored as plain text in VARCHAR or BLOB
Note: for JSON processing, consider UDFs or application-level parsing
Note: CONTAINING and LIKE can do basic JSON value searches
Note: Firebird 5.0+ may add JSON functions (check release notes)
