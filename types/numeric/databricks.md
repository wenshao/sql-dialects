# Databricks SQL: 数值类型

> 参考资料:
> - [Databricks SQL Language Reference](https://docs.databricks.com/en/sql/language-manual/index.html)
> - [Databricks SQL - Built-in Functions](https://docs.databricks.com/en/sql/language-manual/sql-ref-functions-builtin.html)
> - [Delta Lake Documentation](https://docs.delta.io/latest/index.html)


TINYINT: 1 字节，-128 ~ 127
SMALLINT / SHORT: 2 字节，-32768 ~ 32767
INT / INTEGER: 4 字节，-2^31 ~ 2^31-1
BIGINT / LONG: 8 字节，-2^63 ~ 2^63-1

```sql
CREATE TABLE examples (
    tiny_val   TINYINT,                      -- 1 字节
    small_val  SMALLINT,                     -- 2 字节
    int_val    INT,                          -- 4 字节
    big_val    BIGINT                        -- 8 字节
);
```


定点数
DECIMAL(p, s) / NUMERIC(p, s) / DEC(p, s): 最大精度 38
```sql
CREATE TABLE prices (
    price      DECIMAL(10, 2),               -- 10 位总精度，2 位小数
    rate       NUMERIC(5, 4)
);
```


浮点数
FLOAT / REAL: 4 字节单精度 IEEE 754
DOUBLE: 8 字节双精度 IEEE 754
```sql
CREATE TABLE measurements (
    value      FLOAT,                        -- 4 字节单精度
    result     DOUBLE                        -- 8 字节双精度
);
```


布尔
```sql
CREATE TABLE t (active BOOLEAN DEFAULT TRUE);
-- 值: TRUE / FALSE / NULL
```


自增
```sql
CREATE TABLE t (
    id BIGINT GENERATED ALWAYS AS IDENTITY   -- 自增（不可手动设值）
);

CREATE TABLE t2 (
    id BIGINT GENERATED BY DEFAULT AS IDENTITY (START WITH 1 INCREMENT BY 1)
);
```


类型转换
```sql
SELECT CAST('123' AS INT);
SELECT INT('123');                           -- 快捷转换
SELECT BIGINT('123');
SELECT DOUBLE('3.14');
SELECT DECIMAL('123.45');
SELECT CAST('123.45' AS DECIMAL(10, 2));
SELECT TRY_CAST('abc' AS INT);              -- 安全转换，失败返回 NULL
```


数值函数
```sql
SELECT ABS(-5);                              -- 5
SELECT CEIL(3.2);                            -- 4（别名 CEILING）
SELECT FLOOR(3.8);                           -- 3
SELECT ROUND(3.567, 2);                      -- 3.57
SELECT TRUNCATE(3.567, 2);                   -- 3.56
SELECT MOD(10, 3);                           -- 1（或 10 % 3）
SELECT POWER(2, 10);                         -- 1024（或 POW）
SELECT SQRT(144);                            -- 12
SELECT SIGN(-5);                             -- -1
SELECT CBRT(27);                             -- 3（立方根）
SELECT LN(2.718);                            -- ~1
SELECT LOG(10, 100);                         -- 2
SELECT LOG2(8);                              -- 3
SELECT LOG10(100);                           -- 2
SELECT EXP(1);                               -- ~2.718
```


近似计数
```sql
SELECT APPROX_COUNT_DISTINCT(user_id) FROM events;
-- 使用 HyperLogLog++，误差约 5%
```


APPROX_PERCENTILE（近似百分位）
```sql
SELECT APPROX_PERCENTILE(age, 0.5) AS median_age FROM users;
SELECT APPROX_PERCENTILE(age, ARRAY(0.25, 0.5, 0.75)) AS quartiles FROM users;
```


特殊值
```sql
SELECT DOUBLE('Infinity');                   -- 正无穷
SELECT DOUBLE('-Infinity');                  -- 负无穷
SELECT DOUBLE('NaN');                        -- 非数值
```


注意：TINYINT 是有符号的（-128 ~ 127），与 SQL Server 不同
注意：FLOAT 是 4 字节单精度（与 Snowflake/Redshift 不同）
注意：BOOLEAN 原生支持（不是 BIT）
注意：DECIMAL 最大精度 38 位
注意：TRY_CAST 用于安全转换
注意：没有 UNSIGNED 类型
注意：没有 MONEY / SMALLMONEY 类型
